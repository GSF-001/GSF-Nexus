Clone repo: git clone https://github.com/GSF-001/GSF-Nexus.git

 Backend setup:
- cd apps/core-api
- npm install
- cp .env.example .env
- npm run dev

Frontend setup:
- cd ../web-frontend
- npm install
- npm run dev
